# Devicetree Bindings

Place out-of-tree devicetree bindings here.

Future sensor, modem, and board-specific bindings can be added under this directory.

Current local bindings include:

```text
adc/argisense,ads124s08.yaml        - pH front-end 24-bit ADC driver binding
dac/argisense,ad5675-i2c.yaml       - pH ISFET/REFET 8-channel bias DAC driver binding
dac/argisense,gp8302.yaml           - GP8302 current-loop DAC driver binding
power/argisense,lt3582.yaml         - pH +/-5 V analog rail controller placeholder
sensor/argisense,dynament-*.yaml    - Dynament Platinum methane sensor driver binding
sensor/argisense,htu21d.yaml        - HTU21D humidity/temperature driver binding
sensor/argisense,ms5803-05ba.yaml   - MS5803-05BA pressure driver binding
```
